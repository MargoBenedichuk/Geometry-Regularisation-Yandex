### Краткий отчёт

В файлах report-(перечисление параметров модели).txt результаты запусков модели

На графиках тоже описан этап обучения, подробно погружаться в значения параметров считаю бесполезным,
если кратко, 

Image => | ConvNet (Euclid) | => Проекция в гиперболическое пр-во => | Classifier (Hyperbol) | => predicted class


В процессе добавлена регуляризация из статьи ~ $1 / ||x||$ (https://arxiv.org/pdf/2204.08176), можно добавить контрастивный лосс.

Видно, что в запусках report-epoch6-... у нас есть улучшение на 1-2%, когда мы добавляем регуляризацию с коэфицентов 0.05
